# Getting Started

## Installation guide

Ensure your system has installed latest version of yarn or npm.

### Installation via Yarn

```shell
$ yarn add agg-ui-kit
```

### Installation via NPM

```shell
$ npm install agg-ui-kit
```

### Import styles in you entry file

```javascript
import "agg-ui-kit/dist/style.css"
```

### Example usage of the component

```html
<template>
	<div>
		<AButton @click="handleClick" appearance="primary"> Awesome button </AButton>
	</div>
</template>

<script setup lang="ts">
	import { AButton } from "agg-ui-kit"

	const handleClick = () => {
		console.log("Click")
	}
</script>
```
